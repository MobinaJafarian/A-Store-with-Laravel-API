## Setup

```
git clone https://github.com/MobinaJafarian/A-Store-with-Laravel-API.git 
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
```
Then

```
php artisan serve
```
```
npm install
npm run dev
```
